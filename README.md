# ІДЗ 2 — Пошук відділень "Нова пошта" за назвою міста

Проєкт реалізовано за допомогою чистого браузерного JavaScript (HTML5, CSS3, JS Vanilla) та асинхронних запитів до API Нової Пошти.

## ⚠️ Зверніть увагу (Безпека API-ключа)
З міркувань безпеки та вимог до публікації у відкритих репозиторіях, персональний API-ключ було видалено з вихідного коду.

## Як запустити проєкт локально:
1. Клонуйте або завантажте цей репозиторій.
2. Відкрийте файл `script.js` у будь-якому текстовому редакторі.
3. У першому рядку знайдіть змінну `API_KEY` і вставте всередину лапок свій діючий токен, отриманий у бізнес-кабінеті "Нової Пошти":
   ```javascript
   const API_KEY = 'ваш_персональний_токен_тут';
